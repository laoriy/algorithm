# 二叉树

### 树基础知识：

1. 每个节点有多个指针域的称为树，或者可以认为链表是一种特殊的树形解构
2. 将链表的单个指针域改成数组，即为树
3. 每个节点都只有唯一的一个父节点。
4. 二叉树
   - 每个节点度最多为 2
   - 度为 0 的节点比度为 2 的节点多一个（n 个节点，会有 n-1 条边，点的数量 n0+ n1 + n2,边的数量 n1 + 2n2,即为 n0 = n2 + 1）
   - 重要遍历
     - 前序遍历：根左右
     - 中序遍历：左根右
     - 后序遍历：左右根
   - 中序遍历结果 + 前序/后序结果可以还原一颗二叉树
