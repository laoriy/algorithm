### 并查集的基础知识
1. 解决的问题：连通性问题(动态连通性)
    - 传递性：如果A和B连通，B和C连通，那么可以推导出A和C连通
    - 对称性：如果 A和B是相连的，那么B和A是相连的
    - 自反性：A 和 A 是相连的
![image](./images/20160908212440463.png)
2. 维护连通性的关系
    1. 基于染色的思想，一开始所有点的颜色不同
    2. 连接两个点的操作，看成将一种颜色染成另一种
    3. 如果两个点的颜色一样，证明连通，否则不连通
    4. 这种方法叫做并查集的【Quick-find算法】
